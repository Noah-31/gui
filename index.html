The Graphical User Interface

What is the Graphical User Interface?

The Graphical User Interface (GUI) is a way for humans to interact with the computer that uses icons and menus that can be manipulated by the mouse or keyboard. The invention of the GUI made computers easier and more natural to navigate.

Who invented the first GUI?

There was no one inventer of the GUI. The invention of the GUI evolved over time with different inovators each improving on someone else's work. It started when Douglas Engalbart's work led to advances at Xerox. In the early 1700's, several people at Xerox developed the Alto personal computer which was the first computer to use a GUI. This first GUI used windows, icons, and menus. Although Xerox made this product, they didn't sell them.
